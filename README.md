🥗 Nutriq – AI-Powered Nutrition & Calorie Tracking Platform
Nutriq is a full-stack web application that helps users track nutrition, monitor calorie intake, and achieve fitness goals through AI-driven recommendations and personalized diet planning. The platform combines nutrition analytics, food tracking, and intelligent dietary assistance into a single user-friendly experience.

🚀 Features
AI-powered personalized diet plan generation using Google Gemini AI
Daily calorie and macronutrient tracking
Food logging and nutrition analysis
Comprehensive database of 120+ Indian food items with nutritional information
Weight tracking and progress monitoring
Personalized calorie target calculation using BMR and TDEE
Nutrition analytics dashboard with visual insights
Barcode-based food lookup and logging
AI nutrition chatbot for dietary guidance
Premium subscription plans and billing integration
Automated meal reminders and notifications
